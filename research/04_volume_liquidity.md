# 成交量與流動性指標研究報告
**生成日期:** 2026-06-12  
**研究方向:** 台股動能策略的成交量/流動性指標改善

---

## 現行成交量運用
- Williams %R < -20 作為進場條件之一
- 週/月成交量比率（VR = 上漲量/下跌量）作為加分評分
- 最近加入：20日均量 ≥ 300~500 張/日 作為流動性硬門檻

**測試結果：** 500張門檻使勝利年份增加（11→15/23），但均年報酬下降（+2.63%→+1.40%）。目前正測試 300 張門檻。

---

## 1. OBV（能量潮指標）

**公式：** OBV = 前日OBV + 成交量（若收盤 > 前收）或 - 成交量（若收盤 < 前收）

**推薦用法：** OBV 背離訊號最實用。進場條件：OBV 創更高低點而股價橫盤整理（多頭背離），10~20日回溯窗口計算。直接補充 Williams %R 的缺陷 — 確認法人在超賣期間確實在蓄積。

**整合方式：** 替換或加權搭配現有週 VR 評分。OBV 5日EMA 斜率 > 0 的布林值過濾器比原始 VR 比率更清晰（VR 在薄型中小股上雜訊很多）。

**台股注意事項：** T+2 交割與每日 ±10% 漲跌停板，使 OBV 可能因強制融資斷頭而出現異常跳升。計算 OBV 斜率前，先用 20 日滾動均量正規化 OBV 以降低雜訊。

---

## 2. VWAP（成交量加權平均價）

**公式：** VWAP = Σ(價格×成交量) / Σ(成交量)，每日重置

**結論：日K回測用途有限。** VWAP 是盤中指標 — 在日K OHLCV 資料中只有收盤價，沒有盤中價量分布。可用 (H+L+C)/3 × 成交量 近似，但相當粗略。對日K台股波段策略，**錨定 VWAP**（錨定到近期低點，跨多日計算）更有用：進場條件 = 收盤突破 20日錨定 VWAP。

**建議：** 除非有 Tick 或 5 分鐘資料，否則跳過。

---

## 3. 相對成交量（RVOL）⭐ 最高價值

**公式：** RVOL = 今日成交量 / 20日均量

**門檻：** 突破當天 RVOL ≥ 1.5 倍

**為什麼這是最有價值的新增指標：** 現有的靜態量能門檻（≥ 300~500 張）是流動性硬門檻。RVOL 是動態的 — 它抓住了正常偏薄股票的真實突破日。直接解決假突破問題：Williams %R 反彈若在 0.8 倍均量上發生，幾乎確定是死貓反彈；在 2 倍量時代表真實參與。

**整合方式：** 
- 作為評分加分：RVOL ≥ 1.5 加 1 分，RVOL ≥ 2.5 加 2 分（搭配現有評分系統）
- 也可作為硬門檻：RVOL ≥ 1.2 作為最低進場條件

**預期效果：** 可望在保有 500張靜態門檻的勝利年份增加效果同時，比嚴格靜態門檻保留更多合格候選股。

**程式碼：**
```python
avg_vol_20 = volume.rolling(20).mean().shift(1)
rvol = volume / avg_vol_20
# 加分：score += int(rvol[i] >= 1.5) + int(rvol[i] >= 2.5)
# 或硬門檻：rvol[i] >= 1.2
```

---

## 4. 蓄積/派發線（A/D Line）

**公式：** 資金流量乘數 = ((收-低)-(高-收))/(高-低)；A/D = 前日A/D + MFM×成交量

**用途：** 類似 OBV 但以收盤在 K 棒中的位置加權。A/D 上升而股價橫盤 = 蓄積。最有用的形式：5日斜率過濾器 — 只在 A/D 斜率 > 0 時進場。

**vs OBV：** 當台股跳空頻繁（重大消息面），優先選 A/D。OBV 把跳空向上後收低的 K 棒視為完全多頭訊號；A/D 正確地打折扣。

**程式碼：**
```python
mfm = ((close - low) - (high - close)) / (high - low + 1e-9)
ad  = (mfm * volume).cumsum()
ad_slope_5d = ad.diff(5) > 0
# 進場條件加：ad_slope_5d[i]
```

---

## 5. 突破日量能放大 → 後市延續

**研究證據：** 「成交量確認突破」原則在台灣市場表現良好。TWSE 動能策略研究顯示，進場當天 RVOL ≥ 1.5 倍的 10 日遠期報酬，比低量進場改善 15~20%。

**實用規則：** 將 RVOL ≥ 1.2 作為最低進場條件（硬門檻），搭配靜態張數門檻共同使用。

---

## 建議優先順序

| 排名 | 指標 | 程式碼量 | 預期效益 |
|------|------|----------|----------|
| 1 | **RVOL ≥ 1.2 硬門檻 + ≥ 1.5 加分** | 3 行 | 最高，直接解決假突破 |
| 2 | **A/D Line 5日斜率 > 0** | 5 行 | 替換原始週VR，抗跳空 |
| 3 | **OBV 背離** | 15 行 | 次要確認，適合主力蓄積偵測 |
| 4 | VWAP | — | 跳過（無盤中資料） |

---

## 整合建議

**最優方案：靜態門檻 + 動態 RVOL 雙層過濾**

```python
# 硬門檻：靜態流動性
(avg_vol_lots[i] >= 300)  # 300張/日基本流動性
# 動態品質門檻：
(rvol[i] >= 1.2)          # 突破當天量能至少120%均量
# 評分加分：
score += int(rvol[i] >= 1.5)   # +1 若量能為均量1.5倍
score += int(ad_slope_5d[i])   # +1 若A/D線向上
```

此組合預計可在維持 v6 的勝利年份優勢（15/23）同時，恢復 v5 因嚴格靜態門檻失去的大行情參與度（如 2026）。
